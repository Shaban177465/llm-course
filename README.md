# LLM Engineering Projects 🤖

A collection of production-ready LLM applications built with Python and OpenAI API.

## Projects

### 1. 📄 PDF RAG System
Ask questions about any PDF document and get accurate answers.

**Tech:** Python · LangChain · FAISS · OpenAI API

```python
# Ask questions from any PDF
answer = chain.invoke("What is the return policy?")
```

## Tech Stack

- Python 3.x
- OpenAI API (GPT-4o-mini)
- LangChain + LangChain Community
- FAISS Vector Database
- FastAPI
- python-dotenv

## Setup

```bash
git clone https://github.com/Shaban177465/llm-course.git
cd llm-course
pip install -r requirements.txt
```

Add your API key in `.env`:
```
OPENAI_API_KEY=your-key-here
```
